From $A$ we can derive the following set of words: $L_A = \{0^n1^n | n\geq{}1\}$, and $S$ allows for conatenating $A$'s, so $L_S=\{0^{n_1}1^{n_1}0^{n_2}1^{n_2}\dots0^{n_k}1^{n_k} | k\geq{}1, n_1,n_2\dots,n_k\geq{}1\}$.
